# Using this repo

1. **Add a dated folder** (e.g. `2026-03-23-Embedding`) for your experiment.

2. **Write Python** there (often with an AI agent), and split it into cells with `# %% Your title` (same idea as Jupyter / Spyder).

3. **Start Stonesoup** from the repo root: `./start-stonesoup`, then open the UI in the browser and pick the `.py` file to watch.

4. **Run work** from the UI: all cells, a pipeline, or one cell at a time—similar to a notebook.
